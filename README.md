* Version java 25
* Apache Maven 3.8.6
***
Инструкция по установке
=
Для запуска проекта в корневой папке нужно написать команду

`
mvn javafx:run
`
***
Инструкция по пользованию
=
При запуске программы будут доступны 2 кнопки. 
Левая кнопка отвечает за состояние записи, а правая за язык LLM. 
Для начала работы нужно нажать переключатель "Включить запись" и подождать пару секунд, после чего снизу будет отображаться текст, который вы говорите.
Приложение поддерживает 2 языка: русский и английский. Кнопка справа отображает выбранный в данный момент язык, а нажатие на неё поменяет язык.
